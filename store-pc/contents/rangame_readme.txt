cli 에서 게임을 즐기고 싶으신가요??
그럼 업다운 게임으로 즐겨봐요!!

사용하기전에...
이프로그램은 자바로 만들어 졌습니다. 
실행을 위해선 자바가 컴퓨터에 설치되어있어야합니다. 

사용법
1. 프로그램 사용법
	1) 링크에서 프로그램을 받는다.
	2) 터미널, cmd등을 열고 이 프로그램이 있는 디렉토리로 이동한다.
	3) java rangame 이라고 입력한다.
	4) 게임을 즐긴다.

2. 소스코드 사용법
	1) 소스코드를 받고 javac 프로그램도 받습니다.
	2) 컴파일하고 사용합니다
	3) 라이선스
		자유롭게 사용하시고, 수정 재배포 가능합니다.
		다만, 재배포시 원저작자 이름 "박순형" 은 꼭 기재하시기 바랍니다.
